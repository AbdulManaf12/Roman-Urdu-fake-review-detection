Overview

This repository contains the implementation and experimental results for detecting fake reviews in Roman Urdu using novel user behavioral features. The research addresses the challenge of fake review detection in low-resource languages by introducing three types of discriminative features and comprehensive class imbalance handling techniques.

Research Objectives

Novel approach to fake review detection in Roman Urdu using behavioral features.

  • Three-feature engineering approach
  • Class imbalance problem solving
  • Cross-domain performance analysis
  • ML vs DL comparative study

Key Contributions

Novel insights into Roman Urdu fake review detection and performance improvements.

  • Novel user behavioral features (UBF)
  • LSTM-based text generation evaluation
  • Multiple sampling techniques comparison
  • 3% accuracy improvement over baseline

Methodology

Our research follows a comprehensive 6-step methodology for detecting fake reviews in Roman Urdu:

Text Preprocessing

  • Remove special characters and whitespace
  • Single character word removal
  • Roman Urdu stopword filtering
  • Text normalization

Feature Extraction

  • RTF: Review Textual Features
  • RLF: Review Lingual Features
  • UBF: User Behavioral Features
  • Multi-dimensional feature engineering

Resampling Techniques

  • LSTM text generation for RTF
  • Random Oversampling (ROS)
  • Random Undersampling (RUS)
  • SMOTE implementation

Classification Models

  • Gradient Boosting (Best Performance)
  • Random Forest Classifier
  • Support Vector Machines
  • Deep Learning Models
