Abstract

The trend of writing fake reviews has recently increased with the rapid growth of e-commerce websites. This research addresses the challenge of detecting fake reviews in Roman Urdu, a low-resource language, by introducing novel user behavioral features alongside traditional textual and lingual features. We propose a comprehensive methodology that combines machine learning and deep learning approaches to enhance fake review detection.

Our study evaluates multiple experimental configurations including class imbalance handling techniques such as LSTM-based text generation, random undersampling (RUS), and oversampling (ROS). Through empirical comparison of machine learning and deep learning algorithms, we demonstrate a 3% accuracy improvement over baseline studies using Gradient Boosting.

Methodology

Research Methodology: Comprehensive 6-step pipeline for detecting fake reviews in Roman Urdu

Pipeline Overview:

  1. Text Preprocessing: Remove special characters, whitespace, and single character words; Apply Roman Urdu stopword removal
  2. Feature Extraction: Extract Review Textual Features (RTF), Review Lingual Features (RLF), and User Behavioral Features (UBF)
  3. Data Split: 90% for training and validation, 10% for testing
  4. Resampling: Apply hybrid oversampling and undersampling techniques
  5. Classification: Use both Deep Learning and Machine Learning models including Gradient Boosting
  6. Model Evaluation: Assess performance using accuracy, precision, recall, F-score, and confusion matrix
